# 📚 Dự Án Website Nhà Sách - BookStore

Dự án môn học xây dựng giao diện Website bán sách trực tuyến với đầy đủ các tính năng Front-end hiện đại và giao diện Dark Theme thời thượng.

## 🚀 Các tính năng đã hoàn thành (Front-end)
* **Giao diện tối (Dark Theme):** Thiết kế chuẩn UI/UX, dịu mắt và hiện đại.
* **Giỏ hàng thông minh:** Thêm, bớt, tăng giảm số lượng sách và tính tổng tiền trực tiếp qua `localStorage`.
* **Thanh toán nâng cao:** * Form bắt buộc điền thông tin khách hàng (Họ tên, SĐT 10 số).
  * Tích hợp tìm kiếm địa chỉ thông minh (OpenStreetMap API) kèm tự động đo khoảng cách (km).
  * Tính tiền ship tự động theo khoảng cách 
  * Hỗ trợ 2 loại mã giảm giá song song: Giảm giá sách và `FREESHIP` (Miễn phí vận chuyển).

## 📅 Kế hoạch phát triển tiếp theo (Roadmap)
Dự án sẽ được nâng cấp lên mô hình 3 lớp (3-Tier):
1. Thiết lập cơ sở dữ liệu **SQL Server** để lưu trữ thông tin khách hàng, tài khoản và giỏ hàng vĩnh viễn.
2. Xây dựng Server Back-end bằng **Node.js (Express)** để làm cầu nối bảo mật giữa Web và Database.
